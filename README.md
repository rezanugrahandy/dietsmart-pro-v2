# DietSmart Pro - Bonus Buku Diet

Aplikasi web bonus eksklusif untuk pembeli buku diet.

## Fitur

- 🔥 Kalkulator TDEE (Total Daily Energy Expenditure)
- 🍱 Meal Planner dengan database makanan
- 📈 Progress Tracker (berat, lingkar badan)
- 📋 Cheat Sheet makanan sehat
- 💾 Data tersimpan di localStorage (browser)

## Cara Deploy ke GitHub Pages

### 1. Buat Repository di GitHub
1. Buka [github.com](https://github.com)
2. Login dengan akun Anda
3. Klik tombol "+" di atas → "New repository"
4. Isi nama repository: `dietsmart-pro`
5. Pilih "Public"
6. Klik "Create repository"

### 2. Upload File
1. Di halaman repository, klik "uploading an existing file"
2. Drag & drop semua file dari folder ini
3. Klik "Commit changes"

### 3. Aktifkan GitHub Pages
1. Buka tab "Settings" di repository
2. Klik "Pages" di sidebar kiri
3. Di bagian "Source", pilih "Deploy from a branch"
4. Pilih branch "main" dan folder "/ (root)"
5. Klik "Save"
6. Tunggu 2-3 menit, lalu akses URL yang muncul

### 4. URL Aplikasi
Setelah deploy, aplikasi akan tersedia di:
`https://USERNAME.github.io/dietsmart-pro`

## Cara Pakai

1. Buka URL aplikasi
2. Masukkan kode aktivasi dari buku (format: FITME-2026A-XXXXXX)
3. Masukkan nama lengkap
4. Klik "Aktifkan Sekarang"
5. Aplikasi siap digunakan!

## Data Tersimpan di Browser

Semua data (TDEE, meal plan, progress) tersimpan di browser Anda menggunakan localStorage. Data tidak akan hilang selama Anda tidak menghapus cache browser.
